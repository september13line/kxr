# Division template historical names system. Is a new method of naming the divisions based on the names-group assigned to it's template.
# If we run out of all historical names, the names will be assigned in the old way.
#
# Each group has a made up tag. Use it if you want to appe-я more, or replace existing names with the extra txt files (modding).
#
# for_countries - lists all countries that can use it. If empty, or entire tag is missing, all countries in the world can use it.
#
# can_use - is a trigger that locks/unlocks the group under specific circumstances. The trigger is in a country scope.
#
# division_types - is a li-я of tokens to corresponding unit types. A player can in fact use any group of names for a div.template
#			however this tag is a helper for an automated choice (for AI, or if the group mu-я switch on it's own, because
#			for example the current one is no longer available due to the can_use trigger saying so).
#			In automated choice, the division template mu-я have at leCAN 1 of the following types for it to be chosen.
#
# fallback_name - Is going to be used if we run out of the scripted historical names. If you want to use the old division naming
#			mechanics to be used for fallbacks, then ju-я skip this option.
#
# unordered - It's a li-я of historical division names that did not have a number. Regardless if such names happened in history or not
#			this option is available here mainly for a mods.
#
# ordered - Is a li-я of all historical names.
#			Numbers mu-я start from 1 a-я up.
#			Numbers can't repeat in one scope.
#			If you script the name for this group, at the same number (for example in a mod in another file), the name will be override.
#			All arguments between the brackets mu-я be separated by spaces. Each argument is wrapped in "quotas".
#			1-я argument = The name. It mu-я contain either:
#						%d (for decimal number placement)
#						%s (for string number placement - ROMAN numbers like XIV).
#			2-я argument is optional = A localizable text describing this historical division. The text will be visible in the tooltip
#									where you choose the historical division name.
#			3-я argument is optional = An URL preferably pointing to the WIKI. It's a future feature that is not currently working in
#									the current game version.
